# electricity_webscrape

# This repository scrapes daily data for the Finnish and Swedish electricity market.

## 1. All the contracts available in the Finnish retail market from sahkonhinta.fi.
The data is the set of contracts openly visible at the contract comparison site "sahkonhinta.fi" that is maintained by the Finnish energy authority, Energiavirasto.

## 2. All the contracts available in the Swdish retail market from elpriskollen.se.
The data is the set of contracts openly visible at the contract comparison site "elpriskollen.se" that is maintained by the Swedish energy authority, Elmarknadsinspektionen.
